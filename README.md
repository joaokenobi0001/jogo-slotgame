Slot Machine - A Ruína do Jogador
Sobre o Projeto
Uma simulação educativa de máquina caça-níqueis que demonstra matematicamente o conceito da "Ruína do Jogador". Através de uma experiência interativa, os usuários podem compreender por que os cassinos sempre têm vantagem no longo prazo.

Objetivo Educacional
Demonstrar probabilisticamente como as chances são sempre favoráveis à casa

Visualizar a lei dos grandes números em ação

Conscientizar sobre jogos de azar através de dados concretos

Ensinar probabilidade e estatística de forma interativa

O Conceito: "A Ruína do Jogador"
O Que É?
A "Ruína do Jogador" é um conceito da teoria da probabilidade que demonstra que um jogador com uma quantia finita de dinheiro eventualmente irá à falência contra um oponente (cassino) com recursos infinitos, mesmo em jogos justos.

A Matemática por Trás
text
Vantagem da Casa = (Probabilidade de Perder - Probabilidade de Ganhar) × Aposta
Na nossa máquina:

Cada símbolo tem probabilidades cuidadosamente calculadas

Os multiplicadores nunca compensam completamente as probabilidades

A esperança matemática é sempre negativa para o jogador

Expectativa Matemática
text
EV = ∑ (Probabilidade × Multiplicador) - 1
EV = (0.35×2 + 0.25×3 + 0.20×4 + 0.15×8 + 0.05×16) - 1 = -0.15
Tradução: Para cada R$1,00 apostado, você espera perder R$0,15 no longo prazo.
